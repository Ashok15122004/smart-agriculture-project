# 🌾 Smart Agriculture Dashboard

An AI-powered Smart Agriculture Dashboard designed to help farmers monitor crops, analyze environmental conditions, and make data-driven decisions.

## 🚀 Features

* 🌡 Real-time environmental monitoring
* 🤖 AI-powered crop recommendations
* 📈 Interactive dashboard and analytics
* 🔔 Alerts and notifications
* 🌐 Responsive web interface
* 🔗 Backend API integration

## 🛠 Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js

### AI Service

* Python
* Machine Learning

## ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/Ashok15122004/smart-agriculture-project.git
```

### Install Dependencies

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm start
```

AI Service:

```bash
cd ai-service
pip install -r requirements.txt
```
